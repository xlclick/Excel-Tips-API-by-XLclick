**Per andare live basta:**

1. **Push su GitHub** — crea un nuovo repo e fai push di questa cartella

3. **Prima run** → Actions → "Generate API Files" → Run workflow

```
https://YOUR_USERNAME.github.io/xlclick-tips-api/tips.json
https://YOUR_USERNAME.github.io/xlclick-tips-api/tip/today.json

https://YOUR_USERNAME.github.io/xlclick-tips-api/tip/5.json
```
